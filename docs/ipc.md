# Inter-Process Communication

## Header
```
 31            16 15           14 13        12 11          0
+----------------+---------------+------------+-------------+
|      label     | # of channels | # of pages | # of inline |
+----------------+---------------+------------+-------------+
```

## Options
```
 63                                            8 7        0
+-----------------------------------------------+----------+
|                    (reserved)                 |   flags  |
+-----------------------------------------------+----------+
```

### Flags
- `bit 0`: send
- `bit 1`: recv
- `bit 2`: reply
- `bit 3-7`: reserved

## Payloads
### Page
```
 63                                12 11   8 7     3 2    0
+------------------------------------+------+-------+------+
|               offset >> 12         | rsvd |  exp  | type |
+------------------------------------+------+-------+------+
```

#### Types
- `0xb00`: move
- `0xb01`: shared
